generateBoneKeyFrames (bones, callback, startFrame, endFrame, segments)

Parameters:

  • bones a table of Scallion.DomainModels.Components.Bone objects acquiring the keyframes
  • callback the callback function used to set a BoneState object, which accepts an empty BoneState object to be modified and a normalized time parameter
  • startFrame the first frame index of the keyframe sequence
  • endFrame the last frame index of the keyframe sequence
  • segments the number of segments into which this part of the animation is divided; the default is 1
